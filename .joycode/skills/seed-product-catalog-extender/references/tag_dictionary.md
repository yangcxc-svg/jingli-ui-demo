# 送礼标签词典

下列词典用于填 `scenarios / target_people / tags / avoid_for` 字段。**优先使用列出的标准词**，便于推荐过滤命中；语义明显不同的新词可以加，但单条商品同字段词数控制在 3~6 个。

## scenarios（送礼场景）

| 词 | 适用 |
|---|---|
| 见家长 | 第一次拜访伴侣父母 |
| 婚礼 / 婚庆 | 朋友结婚 |
| 乔迁 | 朋友/同事搬新家 |
| 生日 | 通用生日礼 |
| 节日 | 中秋/春节/端午等 |
| 商务 | 客户、合作方、上司 |
| 慰问 | 探望病人、长辈 |
| 升职 / 入职 | 职场场景 |
| 满月 / 周岁 | 婴儿场景 |
| 升学 / 毕业 | 学生场景 |
| 情人节 / 七夕 | 伴侣情感节日 |
| 感恩 | 答谢老师、医生 |

## tags（导购语义标签）

按用途分组，单条商品挑 3~6 个能让用户在前端筛选时看出"为什么推它"的词：

- **分寸/调性**：体面、稳妥、低调、有面子、不显贵、显诚意、轻松、贴心、新潮、传统
- **品质感**：高端、商务、精致、轻奢、入门、性价比、品质感
- **使用价值**：实用、日常、办公、健康、运动、休闲、户外、居家、出差
- **关系氛围**：暖心、文艺、仪式感、惊喜、寓意吉祥、表达关心
- **可送范围**：通用、长辈友好、女生喜欢、男生硬核、孩子爱玩

## avoid_for（不建议送给）

直接写出"具体情境 + 原因"，单条商品 1~3 条：

- "关系不熟"——价格 1500+ 商品避免送陌生客户
- "预算过低"——商品定价已经偏高，预算 200 元以下用户不要推
- "高血压人群"——咖啡、浓茶
- "孕妇"——含酒精商品、含咖啡因商品
- "对花粉过敏"——鲜花类
- "信仰禁忌"——含猪肉、含牛肉等
- "孩子过小"——含小零件、电池玩具
- "习惯极简"——大件装饰、复杂电子产品

## 字段联动建议

写商品时让 `scenarios` × `target_people` × `tags` 形成"画像三角"：

例：`scenarios=["商务", "见家长"]` + `target_people=["长辈", "领导"]` + `tags=["体面", "稳妥", "高端"]`，看起来就是"价位 1500+ 的成熟送礼方案"。如果 scenarios 写"商务"但 tags 全是"新潮 / 学生预算"，明显冲突——这种就要回头调整。
